# YT-DPI Scanner v2.0

В версии 2.0 утилита полностью переписана. Инструмент переведен с последовательного сканирования на многопоточную архитектуру, а также получил полностью новый интерактивный консольный интерфейс. Основной упор в обновлении сделан на скорость работы, детализацию тестируемых протоколов и анализ сети в реальном времени.

### Основные нововведения

* **Многопоточный движок (RunspacePools):** Сканирование больше не выполняется последовательно. Утилита одновременно запускает до 20 независимых фоновых потоков, что сократило общее время проверки всех доменов с ~30 до 2-3 секунд.
* **Детальное разделение протоколов:** Вместо объединения всего TLS-трафика в один результат, теперь отдельно тестируются **открытый HTTP (порт 80)**, **TLS 1.2 (порт 443)** и **TLS 1.3 (порт 443)**. Это позволяет точно определить, на каком уровне шифрования ТСПУ обрывает соединение.
* **Детекция тихих дропов (DRP):** Добавлено точное распознавание ситуаций, когда DPI или файрвол отбрасывает пакеты (blackhole) без отправки ответа. Теперь утилита отличает зависшее по таймауту TLS-рукопожатие (`DRP`) от активного сброса соединения (`RST`).
* **Новый интерактивный интерфейс:** Статичный дашборд с абсолютным позиционированием курсора (без прокрутки консоли). Во время работы отображаются динамические эффекты сканирования: поочередный подбор октетов IP-адреса, перебор Hex-байтов на тестируемых портах и изменяющийся пинг.
* **Живая панель телеметрии:** В правом верхнем углу добавлена панель, которая в реальном времени (в процессе сканирования) обновляет количество активных потоков (JOBS), потребление ОЗУ (RAM) и счетчики: заблокированные домены (BLOCKS), сброшенные пакеты (RST) и успешные подключения (CLEAN).
* **Расширенная сетевая аналитика:** Перед каждым сканированием утилита автоматически определяет и выводит на экран текущий активный **DNS**, **имя провайдера (ISP)**, **страну/город** и ближайший **узел Google CDN**.

### Улучшения и исправления

* **Мгновенная отмена сканирования:** Обработка нажатий `Q` или `ESC` теперь использует асинхронную остановку потоков (`BeginStop`). Интерфейс разблокируется моментально, не дожидаясь сетевых таймаутов зависших соединений.
* **Защита от зависаний консоли:** При запуске утилита через вызов C# API автоматически отключает «Режим быстрого редактирования» (QuickEdit Mode) в Windows. Теперь случайное выделение текста мышью в консоли не будет ставить выполнение скрипта на паузу.
* **Умное определение ОС для TLS 1.3:** Windows 7 и старые сборки Windows 10 не поддерживают TLS 1.3 на уровне системы. Утилита теперь распознает это ограничение ОС и выводит статус `N/A` (Недоступно) вместо ложного срабатывания `RST` (Блокировка).
* **Встроенная справка:** Добавлена кнопка `[H]`, открывающая мини-гайд с расшифровкой статус-кодов (OK, RST, DRP, N/A, FAIL) и вердиктов прямо внутри приложения, без необходимости переходить в браузер.
* **Устранено моргание экрана:** Механизм рендеринга интерфейса оптимизирован. Данные обновляются поверх старых значений без полной очистки консоли (`Clear-Host`), что убрало неприятное мерцание при повторных запусках сканирования.

### Расшифровка статус-кодов
* **OK:** Соединение установлено, TLS-рукопожатие завершено успешно.
* **RST:** Соединение принудительно разорвано (TCP Reset) оборудованием DPI/ТСПУ.
* **DRP:** Соединение "тихо" отброшено (Blackhole), TLS-рукопожатие прервано по таймауту.
* **N/A:** Протокол не поддерживается вашей ОС (например, TLS 1.3 на Windows 7).
* **--** / **FAIL:** Ошибка сокета или полная недоступность IP-адреса.

### Установка и использование
1. Скачайте файл **`YT-DPI-Scanner-v2.0.bat`** из раздела Assets ниже.
2. Запустите файл (права администратора не требуются).
3. Нажмите `[ENTER]` для запуска сканирования или `[H]` для чтения справки.

*Отказ от ответственности: Данный инструмент предназначен исключительно для диагностических и образовательных целей. Он помогает определить методы блокировки, но сам по себе не является средством обхода ограничений.*
