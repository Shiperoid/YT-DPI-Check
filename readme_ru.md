# YT-DPI Check

[[English language](README.md)]

### Описание
**YT-DPI Check** — это легкая утилита на PowerShell, предназначенная для выявления точных причин проблем с доступом к YouTube. Скрипт проводит многоуровневый анализ сети, чтобы отличить обычные сбои соединения от целенаправленной фильтрации трафика через DPI (ТСПУ).

### Основные возможности
*   **Проверка целостности DNS:** Обнаруживает ошибки разрешения имен или подмену IP.
*   **Проверка TCP соединения:** Проверяет доступность серверов Google/YouTube на порту 443.
*   **Тест TLS/SNI Handshake:** Симулирует защищенное соединение для обнаружения вмешательства DPI (сброс пакетов при обнаружении имени домена).
*   **Поиск локального CDN:** Автоматически находит и тестирует ближайший узел Google Global Cache (GGC).
*   **Компактный UI:** Таблица адаптирована под стандартный размер окна консоли (80 символов).
*   **Авто-диагноз:** Скрипт анализирует результаты и выводит итоговое заключение.

### Как использовать
1.  Скачайте файл скрипта `YT-DPI-Check.ps1`.
2.  Нажмите правой кнопкой мыши на файл и выберите **"Выполнить с помощью PowerShell"**.
3.  *Если система блокирует запуск, выполните в терминале:*
    ```powershell
    Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
    .\YT-DPI-Check.ps1
    ```

### Интерпретация сокращений (Legend)
*   **OK**: Успешное соединение.
*   **FL** (Failed): Ошибка подключения (порт закрыт или сброшен).
*   **DR** (Dropped): Пакет ушел, но ответ не пришел. Прямой признак работы DPI.
*   **--** (Skipped): Тест пропущен (если предыдущий этап завершился ошибкой).

### Интерпретация вердиктов (Result)
| Результат | Значение |
| :--- | :--- |
| **AVAILABLE** | Соединение полностью доступно. |
| **DPI BLOCK** | Обнаружена фильтрация по SNI (блокировка на уровне DPI). |
| **IP BLOCK** | IP-адрес недоступен. Возможна блокировка по IP. |
| **DNS ERROR** | Не удалось разрешить имя домена в IP-адрес. |
