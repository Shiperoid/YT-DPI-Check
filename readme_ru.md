# YT-DPI Diagnostic Tool

[[English language](readme.md)]


### Описание
**YT-DPI Diagnostic Tool** — это легкая утилита на PowerShell, предназначенная для выявления точных причин проблем с доступом к YouTube. Скрипт проводит многоуровневый анализ сети, чтобы отличить обычные сбои соединения от целенаправленной фильтрации трафика через DPI (ТСПУ).

### Основные возможности
*   **Проверка целостности DNS:** Обнаруживает перехват или подмену DNS (например, когда домен резолвится в `127.0.0.1`).
*   **Проверка TCP соединения:** Проверяет, доступны ли IP-адреса сервисов Google/YouTube на порту 443.
*   **Тест TLS/SNI Handshake:** Симулирует защищенное соединение для обнаружения вмешательства DPI.
*   **Поиск локального CDN:** Автоматически находит и тестирует ближайший к вам узел Google Global Cache (GGC).
*   **Поддержка старых ОС:** Корректно работает в Windows 7, 10 и 11 (учитывает особенности TLS 1.2/1.3).
*   **Логика повторов:** Встроенная система повторных попыток для исключения ложных результатов из-за нестабильности сети.

### Как использовать
1.  Скачайте файл скрипта `YT-DPI-Test.ps1`.
2.  Нажмите правой кнопкой мыши на файл и выберите **"Выполнить с помощью PowerShell"**.
3.  *Дополнительно:* Если система блокирует запуск, выполните в терминале:
    ```powershell
    Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
    .\YT-DPI-Test.ps1
    ```

### Интерпретация результатов
| Результат | Значение |
| :--- | :--- |
| **AVAILABLE** | Соединение полностью доступно. |
| **DPI BLOCK (SNI)** | Провайдер видит имя "youtube.com" внутри пакета и сбрасывает его. Типичная блокировка DPI. |
| **IP BLOCK** | IP-адрес полностью недоступен. Провайдер заблокировал адрес на уровне маршрутизации. |
| **DNS SPOOFED** | Ваш DNS-провайдер возвращает поддельные IP-адреса (например, 127.0.0.1). |
| **ERROR** | Общая сетевая ошибка или превышение времени ожидания. |
